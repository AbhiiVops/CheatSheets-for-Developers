# Chrome DevTools Keyboard Shortcuts for Developers

## Keyboard shortcuts for opening DevTools

| Action | Mac  | Windows/Linux  | 
| :----: | :--: | :------------: |
| Open the ___Console___ panel | `cmd(‚åò)+option(‚å•)+j` | `ctrl+shift+j` | 
| Open the ___Elements___ panel | `cmd(‚åò)+shift+c` or<br />`cmd(‚åò)+option(‚å•)+c`   | `ctrl+shift+c` | 
| Open whatever panel you used last | `cmd(‚åò)+option(‚å•)+i` | `F12` or<br /> `ctrl+shift+i` |
